Flowershop - это интернет-магазин для услуг по изготовлению букетов и цветочных композиций.

Front-end часть выполнена на основе html шаблона, взятого с сайта templated.co и отредактирована под требуемый вид.

Back-end часть выполнена на чистом PHP без использования фреймворков.
Сайт выполнен на основе паттерна MVC с использованием ООП, для работы с базой данных MySQL используется расширение PDO, 
для добавления товаров в корзину используется AJAX, JQuery.
